Welcome to my first data analysis project! This interactive Coffee Sales Dashboard was built using Microsoft Excel to explore and visualize coffee sales data from 2019 to 2022 across four major cities.

Key Features:
Dynamic Filters: Analyze data by roast type (Dark, Medium, Light), coffee size (weight), and loyalty card usage.

Trend Analysis: A line graph showing sales performance over time.

Customer Insights: A bar chart of the Top 5 Customers based on total sales.

Geographical Overview: Country-wise sales breakdown using a clear bar visualization.
